Seneca Plugins

Use these these plugins to kick-start your application development. They provide the basic business logic for many common use cases, like user accounts, shopping carts, or administration. You can customize their behaviour by overriding their actions.

For the list of data stores, see below.

  • Web

    Provides a HTTP API builder for actions. Included by default.
  • Transport

    Provides a micro-service communication layer. Included by default.
  • User

    Provides the business logic for user accounts: login, logout, password encryption, resets, etc.
  • Auth

    Provides the business logic for authentication via HTTP.
  • JSON REST API

    Provides a JSON REST API for data entities.
  • Data Editor

    Data editing interface, inspired by Django.
  • Admin

    Provides an administration console.
  • Email

    Provides email delivery business logic, including templating.
  • Account

    Provides a user account system (there can be multiple users in account).
  • Project

    Provides a project system. Accounts own projects.
  • Perm

    Provides a low-level permissions system for data and actions.
  • Wizard

    Provides business logic for wizard interfaces.
  • VCache

    Provides versioned caching for data entities.
  • Cart

    Provides shopping cart business logic.
  • Pay

    Provides payments business logic.
  • CMS

    Provides a simple content management system.
  • Settings

    Settings for user accounts.
