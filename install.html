Download and Install

Current Version: 0.5.9

Seneca is distributed as an npm module. To download and install, open a terminal or command line prompt and go to your project folder. Run the command:

{% highlight bash %} $ npm install seneca {% endhighlight %}

In your code, you can import the Seneca module, and create a new object instance, with this code:

{% highlight js %} var seneca = require('seneca')() {% endhighlight %}

Read the Getting Started Guide to start using Seneca in your app.

You can also clone Seneca directly from GitHub: http://github.com/rjrodger/seneca. Do this if you need to use a development branch.
